# Game of Life

## Run the binary
```
git clone https://github.com/glitzflitz/gameoflife
cd gameoflife
chmod +x run.sh
./run.sh
```

## Build
```
git clone https://github.com/glitzflitz/gameoflife
cd gameoflife
cargo install bootimage
cargo install cargo-xbuild
rustup component add rust-src
#Install qemu
cargo xrun
```
